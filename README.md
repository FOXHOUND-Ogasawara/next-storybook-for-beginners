# Next storybook

Next.js + Storybook + Tailwind CSS + TypeScript

### Prerequisites

- [Node.js](https://nodejs.org/) v20.11.1 or higher
- [pnpm](https://pnpm.io/ja/installation) v8.15.0 or higher

## Project setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Storybook

```bash
pnpm storybook
```

## Build

```bash
pnpm build
```
